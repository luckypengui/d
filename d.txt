Neon Dash

### Das Spielkonzept

* **Ziel:** Du bist ein blaues Quadrat und musst den roten Bällen ausweichen, die von oben herabfallen.
* **Steuerung PC:** Pfeiltasten (Links/Rechts) oder Mausbewegung.
* **Steuerung Handy:** Tippe/Halte auf die linke oder rechte Bildschirmhälfte.
* **Besonderheit:** Es wird schneller, je mehr Punkte du hast.

### Der Code (index.html)

Kopiere den folgenden Code einfach in eine leere Textdatei und speichere sie als `index.html`.

```html
<!DOCTYPE html>
<html lang="de">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no">
    <title>Neon Dash</title>
    <style>
        /* Grundlegendes Styling für Vollbild ohne Scrollen */
        body {
            margin: 0;
            padding: 0;
            overflow: hidden;
            background-color: #111;
            font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
            touch-action: none; /* Verhindert Zoom/Scroll auf Touchscreens */
            user-select: none; /* Verhindert Textauswahl */
        }

        canvas {
            display: block;
        }

        /* UI Overlay */
        #ui-layer {
            position: absolute;
            top: 0;
            left: 0;
            width: 100%;
            height: 100%;
            pointer-events: none; /* Klicks gehen durch das UI durch (außer Buttons) */
            display: flex;
            flex-direction: column;
            justify-content: center;
            align-items: center;
            text-align: center;
        }

        #score-display {
            position: absolute;
            top: 20px;
            left: 20px;
            color: white;
            font-size: 24px;
            font-weight: bold;
            text-shadow: 0 0 10px rgba(0, 255, 255, 0.7);
        }

        #start-screen, #game-over-screen {
            background: rgba(0, 0, 0, 0.85);
            padding: 40px;
            border-radius: 15px;
            border: 2px solid #00ffff;
            box-shadow: 0 0 20px rgba(0, 255, 255, 0.3);
            pointer-events: auto;
            transition: opacity 0.3s;
        }

        h1 {
            color: #00ffff;
            margin: 0 0 10px 0;
            text-transform: uppercase;
            letter-spacing: 3px;
        }

        p {
            color: #ccc;
            margin-bottom: 25px;
        }

        button {
            background: #ff0055;
            color: white;
            border: none;
            padding: 15px 30px;
            font-size: 18px;
            border-radius: 30px;
            cursor: pointer;
            font-weight: bold;
            text-transform: uppercase;
            box-shadow: 0 0 15px rgba(255, 0, 85, 0.6);
            transition: transform 0.1s;
        }

        button:active {
            transform: scale(0.95);
        }

        .hidden {
            display: none !important;
        }

        /* Hilfslinien für Touch-Steuerung (optional) */
        .controls-hint {
            position: absolute;
            bottom: 20px;
            width: 100%;
            text-align: center;
            color: rgba(255,255,255,0.2);
            font-size: 12px;
            pointer-events: none;
        }
    </style>
</head>
<body>

    <div id="score-display">Score: 0</div>

    <canvas id="gameCanvas"></canvas>

    <div id="ui-layer">
        <div id="start-screen">
            <h1>Neon Dash</h1>
            <p>PC: Pfeiltasten / Maus<br>Handy: Tippe Links / Rechts</p>
            <button onclick="startGame()">Starten</button>
        </div>

        <div id="game-over-screen" class="hidden">
            <h1 style="color: #ff0055;">Game Over</h1>
            <p>Dein Score: <span id="final-score">0</span></p>
            <button onclick="startGame()">Nochmal</button>
        </div>
        
        <div class="controls-hint" id="controls-hint">
            Links tippen &larr;  &rarr; Rechts tippen
        </div>
    </div>

    <script>
        const canvas = document.getElementById('gameCanvas');
        const ctx = canvas.getContext('2d');
        
        // Screens
        const startScreen = document.getElementById('start-screen');
        const gameOverScreen = document.getElementById('game-over-screen');
        const scoreDisplay = document.getElementById('score-display');
        const finalScoreSpan = document.getElementById('final-score');
        const controlsHint = document.getElementById('controls-hint');

        // Game State
        let gameRunning = false;
        let score = 0;
        let animationId;
        let lastTime = 0;
        let enemySpawnTimer = 0;

        // Player
        const player = {
            x: 0,
            y: 0,
            width: 40,
            height: 40,
            speed: 400, // Pixel pro Sekunde
            color: '#00ffff'
        };

        // Inputs
        const keys = {
            ArrowLeft: false,
            ArrowRight: false
        };
        const touchInput = {
            active: false,
            x: 0
        };

        // Enemies
        let enemies = [];
        let baseEnemySpeed = 200;

        // Setup Canvas Größe
        function resize() {
            canvas.width = window.innerWidth;
            canvas.height = window.innerHeight;
            player.y = canvas.height - 100;
            // Spieler zentrieren beim Resize, wenn Spiel nicht läuft
            if (!gameRunning) player.x = canvas.width / 2 - player.width / 2;
        }
        window.addEventListener('resize', resize);
        resize();

        // Input Event Listeners (Tastatur)
        window.addEventListener('keydown', (e) => keys[e.code] = true);
        window.addEventListener('keyup', (e) => keys[e.code] = false);

        // Input Event Listeners (Touch)
        window.addEventListener('touchstart', (e) => {
            touchInput.active = true;
            touchInput.x = e.touches[0].clientX;
        }, {passive: false});

        window.addEventListener('touchmove', (e) => {
            touchInput.x = e.touches[0].clientX; // Für Drag-Bewegung
            // Verhindert das Neuladen durch Ziehen nach unten (Chrome Android)
            e.preventDefault(); 
        }, {passive: false});

        window.addEventListener('touchend', () => {
            touchInput.active = false;
        });

        // Start Funktion
        function startGame() {
            startScreen.classList.add('hidden');
            gameOverScreen.classList.add('hidden');
            controlsHint.classList.add('hidden'); // Hinweise ausblenden beim Spielen
            
            gameRunning = true;
            score = 0;
            enemies = [];
            baseEnemySpeed = 200;
            player.x = canvas.width / 2 - player.width / 2;
            
            scoreDisplay.innerText = `Score: 0`;
            
            lastTime = performance.now();
            requestAnimationFrame(gameLoop);
        }

        // Game Over Funktion
        function gameOver() {
            gameRunning = false;
            finalScoreSpan.innerText = Math.floor(score);
            gameOverScreen.classList.remove('hidden');
        }

        // Update Logik
        function update(deltaTime) {
            // 1. Spieler Bewegung
            let moveX = 0;
            
            // Tastatur
            if (keys.ArrowLeft) moveX = -1;
            if (keys.ArrowRight) moveX = 1;

            if (moveX !== 0) {
                player.x += moveX * player.speed * deltaTime;
            }

            // Touch Logik: Wenn Touch aktiv ist, bewege Spieler in Richtung Finger
            // Wir teilen den Bildschirm in zwei Hälften für einfache Steuerung
            if (touchInput.active) {
                if (touchInput.x < window.innerWidth / 2) {
                    player.x -= player.speed * deltaTime;
                } else {
                    player.x += player.speed * deltaTime;
                }
            }

            // Bildschirmgrenzen
            if (player.x < 0) player.x = 0;
            if (player.x + player.width > canvas.width) player.x = canvas.width - player.width;

            // 2. Gegner Spawnen
            enemySpawnTimer += deltaTime;
            // Spawn Rate wird schneller bei höherem Score
            let spawnRate = Math.max(0.2, 1.0 - (score / 5000)); 
            
            if (enemySpawnTimer > spawnRate) {
                const size = Math.random() * 30 + 20; // Zufällige Größe 20-50px
                enemies.push({
                    x: Math.random() * (canvas.width - size),
                    y: -size,
                    size: size,
                    speed: baseEnemySpeed + (Math.random() * 100) + (score / 10) // Wird schneller
                });
                enemySpawnTimer = 0;
            }

            // 3. Gegner Bewegen & Kollision
            for (let i = enemies.length - 1; i >= 0; i--) {
                let enemy = enemies[i];
                enemy.y += enemy.speed * deltaTime;

                // Kollision (Einfache Rechteck-Kollision)
                if (
                    player.x < enemy.x + enemy.size &&
                    player.x + player.width > enemy.x &&
                    player.y < enemy.y + enemy.size &&
                    player.y + player.height > enemy.y
                ) {
                    gameOver();
                }

                // Entfernen wenn unten raus
                if (enemy.y > canvas.height) {
                    enemies.splice(i, 1);
                    score += 10;
                    scoreDisplay.innerText = `Score: ${score}`;
                    
                    // Schwierigkeit erhöhen
                    baseEnemySpeed += 2;
                }
            }
        }

        // Zeichnen
        function draw() {
            // Hintergrund leeren
            ctx.fillStyle = '#111'; // Dunkler Hintergrund
            ctx.fillRect(0, 0, canvas.width, canvas.height);

            // Gitter-Effekt (Retro Look)
            ctx.strokeStyle = '#222';
            ctx.lineWidth = 2;
            ctx.beginPath();
            for(let x=0; x<canvas.width; x+=50) { ctx.moveTo(x,0); ctx.lineTo(x,canvas.height); }
            for(let y=0; y<canvas.height; y+=50) { ctx.moveTo(0,y); ctx.lineTo(canvas.width,y); }
            ctx.stroke();

            // Spieler zeichnen (Neon Glow)
            ctx.shadowBlur = 20;
            ctx.shadowColor = player.color;
            ctx.fillStyle = player.color;
            ctx.fillRect(player.x, player.y, player.width, player.height);
            ctx.shadowBlur = 0; // Reset Shadow

            // Gegner zeichnen
            ctx.fillStyle = '#ff0055';
            for (let enemy of enemies) {
                ctx.beginPath();
                ctx.arc(enemy.x + enemy.size/2, enemy.y + enemy.size/2, enemy.size/2, 0, Math.PI * 2);
                ctx.fill();
            }
        }

        // Game Loop
        function gameLoop(timestamp) {
            if (!gameRunning) return;

            let deltaTime = (timestamp - lastTime) / 1000;
            lastTime = timestamp;

            update(deltaTime);
            draw();

            requestAnimationFrame(gameLoop);
        }
        
    </script>
</body>
</html>

```

### Warum dieses Spiel gut funktioniert:

1. **Responsivität:** Das Canvas passt sich automatisch an (`window.innerWidth/Height`). Egal ob Hochformat (Handy) oder Querformat (PC), es füllt den Bildschirm.
2. **Touch-Optimierung:** Ich habe CSS-Regeln (`touch-action: none`) hinzugefügt, die verhindern, dass der Browser zoomt oder scrollt, wenn man wild auf das Handy tippt.
3. **Steuerung:**
* **PC:** Die Pfeiltasten fühlen sich sehr präzise an.
* **Smartphone:** Ich habe eine "Split-Screen"-Logik eingebaut. Tippst du irgendwo auf die **linke** Seite, fährt der Spieler nach links. Tippst du **rechts**, fährt er nach rechts. Das ist viel einfacher zu bedienen als kleine virtuelle Buttons.


4. **Technik:** Es nutzt `requestAnimationFrame` für flüssige 60 FPS (oder höher auf modernen Screens) und berechnet Bewegungen basierend auf der Zeit (`deltaTime`), damit das Spiel auf schnellen und langsamen Geräten gleich schnell läuft.

### Anleitung zum Testen:

1. Erstelle einen Ordner auf deinem PC.
2. Erstelle darin eine Datei namens `index.html`.
3. Kopiere den Code oben hinein.
4. Öffne die Datei mit Chrome, Firefox oder Edge.
5. Um es auf dem **Handy** zu testen: Wenn du im gleichen WLAN bist, kannst du die Datei auf einem lokalen Webserver hosten (z.B. mit der VS Code Extension "Live Server") und die IP-Adresse im Handy eingeben. Oder du lädst die Datei einfach auf GitHub Pages oder Netlify hoch (Drag & Drop).
